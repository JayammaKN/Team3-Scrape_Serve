# Team3 - Scrape & Serve

This project collects product or content URLs, scrapes data, filters it, and writes results to a database.

## Project structure

- `src/` contains the main pipeline modules
- `sql/` contains SQL scripts for database-related tasks

## Getting started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the main script:
   ```bash
   npm start
   ```

## Notes

Adjust the configuration in `src/config.js` as needed for your scraping target and database setup.
